Play on Windows

  1. Press Win + R to open the "Run" dialog.
  2. Type the following path into the textbox and press Enter:
    C:\Windows\System32\drivers\etc
  3. Find the file named hosts.
  4. Right-click the `hosts` file, and open it with an editor like Notepad (you may need to run Notepad as an Administrator).
  5. Add the following two lines to the very bottom of the file: 
    167.71.202.174 www.growtopia1.com
167.71.202.174 www.growtopia2.com
  6. Save the file and close the editor.
  7. You can now log into the game. Enjoy!
