Play on macOS

  1. Open Finder, click "Go" in the menu bar, then select "Go to Folder...".
  2. Enter the following path and click "Go":
    /private/etc/hosts
  3. Copy the `hosts` file to your Desktop.
  4. Open the copied `hosts` file on your Desktop with a text editor.
  5. Add the following two lines to the end of the file: 
    167.71.202.174 www.growtopia1.com
167.71.202.174 www.growtopia2.com
  6. Save the file and move it back to `/private/etc/`, replacing the original file (you will need to enter your password).
  7. Start Growtopia and enjoy!
