Play on Android

Method 1: Firefly APK (Recommended)

  1. Download Firefly APK here: Firefly.apk (v5.05)
  2. Open the Firefly APK after installing.
  3. Click on "Start".
  4. The app will automatically open Growtopia and connect you to the Firefly Private Server.

Method 2: Virtual Hosts

  1. Download the Virtual Hosts App: virtual_host.apk
  2. Install the Virtual Hosts app.
  3. Download the Hosts File: firefly.txt
  4. Open Virtual Hosts and click on "Select/Reselect hosts file".
  5. Select the `firefly.txt` file you just downloaded.
  6. Click the large toggle switch to turn it on (it should turn green).
  7. Open Growtopia and connect!

Method 3: PowerTunnel

  1. Download and install the PowerTunnel App.
  2. In the app, add this URL for the "Hosts File URL": https://gtps.firefly.my.id/download/firefly
  3. Change the "Hosts file update period" to "On start".
  4. Go back to the main menu and click "Connect".
  5. Once connected, open Growtopia and log in.
